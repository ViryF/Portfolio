Tic-Tac-Toe

This was the first application I ever built. It was built with HTML5, Vanilla Javascript and CSS3. This application was deployed with Vercel


It's my version of Tic-Tac-Toe. It's still a work in progress, but feel free to check out my Github repository. The game starts off with an empty board and signals that it's "Player X's" turn. After the user clicks on any section, the turn changes to "Player O" and the any one of the remaining squares can be filled. There is a specific set of winning combinations. If one of them is met, a message will appear announcing the winner above the Play Again button. If no one wins, there will be a "DRAW" message.

Melting Ice Cream

This application was built using HTML5, Vanilla, Javascript, CSS3 and deployed with Surge.


This is my version of the game: Hangman. The player can reach a total of 6 incorrect guesses before the game ends. If a guess is correct, the letter guessed appears on the appropriate field. If the letter is incorrect, a part of the ice cream melts and the current mistake count increases. If the player wins, they see a winning statement and the alphabet becomes unavailable.
